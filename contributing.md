**Hướng dẫn contribute cho ứng dụng.**

 1. Fork repository
 2. Clone repository về máy
 3. Tạo branch mới (nếu cầu)
 4. Commit trên branch vừa tạo (hoặc default: master)
 5. Tạo pull request (Cái này t sẽ làm)
 6. Merge code ( Cái này t sẽ làm)
